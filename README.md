# Nami [![CircleCI](https://circleci.com/gh/cfx-cv/nami.svg?style=svg)](https://circleci.com/gh/cfx-cv/nami)
Go-Feed microservice providing route maps, her dream is to draw the entire world map

## Getting Started

### Installation

Using `docker`:

1. Install `docker` and `docker-compose`.
2. `cp env.sample .env` and set values accordingly.
3. `docker-compose up`.
